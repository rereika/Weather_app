# テーマ
地域を選択すると、天気がわかるサービス

# 誰のどんな課題を解決するのか？
雨がいつ降るかをすぐに知りたい、自転車通勤、洗濯好き、せっかちに愛用される天気アプリ

# 機能要件
P0：必須<br>
P1：ここまで実装することを想定<br>
P2：可能であれば実装したい
- LP(初期画面)
    - プレースホルダー：「地域名や住所を入力」(P0)
    - ユーザーが文字を入力すると、住所や地域名が候補としてリアルタイムに表示(P1)
    - 現在地取得ボタン(P0)
- 天気の表示
    - 地域名、現在の天気、気温(P0)
    - 1時間ごとの天気(P0)
    - 次いつ雨降るか(P0)
